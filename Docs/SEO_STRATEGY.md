# JodTod - SEO Strategy

No marketing budget, so SEO is the primary traffic source.

---

## 1. Technical SEO

### Page Speed
- Blade pages for public content (fast server-side render)
- Minify CSS/JS in production
- Optimize images (WebP format, lazy loading)
- Enable gzip compression
- Browser caching headers

### Meta Tags (Every public page must have)
- `<title>` - unique, 50-60 chars, keyword included
- `<meta name="description">` - unique, 150-160 chars
- `<meta name="robots" content="index, follow">`
- Canonical URL
- Open Graph tags (og:title, og:description, og:image, og:url)
- Twitter Card tags

### Structured Data (Schema.org)
- **Home page:** WebApplication schema
- **Blog posts:** BlogPosting / Article schema
- **About page:** Organization schema
- **Tools page:** SoftwareApplication schema
- **FAQ sections:** FAQPage schema

### Technical Requirements
- Auto-generate `sitemap.xml` (update on new blog post)
- `robots.txt` configured properly
- SSL certificate (HTTPS)
- Mobile-friendly (responsive)
- Clean URL structure (no query params for public pages)
- 404 page with helpful links
- Breadcrumb navigation
